##

#### 1. Clone Repository

```
 git clone
 cd sistemasjid
 composer install
 npm install
 copy .env.example .env
```

#### 2. Instalasi Website

```
 php artisan key:generate
```

#### 3. Menjalankan Website

```
 php artisan serve
```
